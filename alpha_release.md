# Instructions
### 1. Clone the repository:
```bash
git clone git@github.com:dharmik529/CampusAgile.git
cd CampusAgile
```

### 2. Install Node.js:
* Ensure you have Node.js installed on your machine.
* If not, you can download it from https://nodejs.org/.

### 3. Install NestJS
```bash
npm install -g @nestjs/cli
```

### 4. Build and Test
#### Front-end
```bash
cd client
npm install
npm run dev
```

#### Back-end
```bash
cd api
npm install
nest start
```
